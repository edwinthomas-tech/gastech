The Problem

Traditional LPG cylinders provide no clear indication of remaining gas. Manual estimation leads to inconvenience, disrupted routines, and safety concerns—especially in hostels and shared kitchens.

Our Solution

GASTECH integrates a load cell with an ESP32 microcontroller to continuously measure cylinder weight. The system calculates gas percentage, syncs data to the cloud, and presents insights through a modern web application.

How It Works

The load cell measures weight changes in the LPG cylinder and feeds data to the ESP32. The microcontroller processes this data and uploads the gas percentage to Firebase in real time.

The web dashboard retrieves this data instantly, displaying live gas levels, estimated remaining days, and refill alerts in a clean interface.

Designed for Real Life

From homes to hostels and shared kitchens, GASTECH ensures uninterrupted cooking by enabling timely refill planning and reducing dependency on guesswork.

Technology Stack

The system uses ESP32, HX711 load cell amplifier, and Firebase Realtime Database. The frontend is built using modern HTML, CSS, and JavaScript for responsiveness and real-time updates.

Future Enhancements

Future plans include SMS and mobile notifications, consumption analytics, multi-cylinder support, and intelligent refill recommendations.
